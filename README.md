# Iconic

A production-grade distributed salon booking platform built with Spring Boot 3.5 microservices.

## Features

- JWT authentication via Keycloak
- event-driven notifications with RabbitMQ
- real-time notification using WebSocket
- Razorpay/Stripe payment integration
- distributed tracing with Zipkin
- observability via Prometheus + Grafana
- Redis caching
- Full deployment on Kubernetes with Helm
